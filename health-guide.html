 운동기구 사용방법

[ 숄더 프레스 운동방법 ]

- 운동부위 : 어깨 전면 삼각근과 회전근개

- 운동횟수 : 한 세트당 12~15회, 3~5 세트

- 운동방법
  1. 자신 근력에 맞는 무게 추를 조정한다.
  2. 팔꿈치가 수직이 되게 만들어주고 올릴 때 숨을 내쉬고 내릴 때들이 마셔준다.
  3. 허리 또는 등이 말리지 않게 주의해야 한다.

[ 랫풀 다운 운동방법 ]

- 운동부위 : 등근육과 이두근

- 운동횟수 : 한 세트당 12~15회, 3~5 세트

- 운동방법
  1. 자신 근력에 맞는 무게 추를 조정한다.
  2. 밑으로 당길 때 호흡을 내쉬고, 올려줄 때 호흡을 들이 마십니다.
  3. 허리가 앞으로 말리지 않게 주의하며 가슴을 벌려주고 고개를 들어줍니다.

[ 레그 프레스 운동방법 ]

- 운동부위 : 대퇴사 두근(허벅지 앞),둔근(엉덩이),햄스트링(허벅지 뒤)

- 운동횟수 : 한 세트당 15회 이상, 3~5 세트

- 운동방법
  1. 자신 근력에 맞는 바벨 무게를 조정한다.
  2. 상체와 엉덩이를 머신에 밀착 고정한다.
  3. 발은 어깨너비 또는 조금 더 넓게 위치한다.
  4. 발꿈치에 힘을 주어 밀고 호흡을 내뱉는다.
    (무릎을 완전히 펴지않아야한다. 부상위험)
  5. 신전될 때는 숨을 마시며 다리가 90도 될 때 까지 구부린다.

 신체부위별 추천 운동

[ 하이퍼 익스텐션 운동방법 ]

- 운동부위 : 등근육과 뒤허벅지

- 운동횟수 : 한 세트당 15~20회, 5 세트

- 운동방법
  1. 밑 부분 발패드에 발목을 걸고 허벅지가 윗부분 패드에 닿도록 합니다.
  2. 자신의 골반 아래쪽에 윗부분 패드가 오도록 높낮이를 조절해준다.
  3. 팔을 X자 모양으로 겹쳐 가슴에 붙이고 허리를 일자로 펴준다.
  4. 천천히 허리를 앞으로 굽힙니다. 이때 등은 계속 일자상태로 유지해준다.
  5. 다시 천천히 올라오면서 허리가 휘어지지 않도록 주의한다.

[ 체스트 프레스 운동방법 ]

- 운동부위 : 가슴근육, 팔뚝살 슬림하게~

- 운동횟수 : 한 세트당 10~12회, 3~4 세트

- 운동방법
  1. 자신에게 맞는 무게를 설정한다.
  2. 팔 힘으로 핸들을 밀어낼 때 호흡을 내쉬고, 팔을 내릴 때 호흡을 들이 마십니다.
  3. 10~12개 실시 후, 30~40초 쉬어 줍니다.
  4. 위 내용을 3~4 세트 반복해 줍니다.

[ 힙 어브 덕터 운동방법 ]

- 운동부위 : 측명 엉덩이 근육(중둔근,소둔근), 애플힙

- 운동횟수 : 한 세트당 15회, 3~5 세트

- 운동방법
  1. 다리를 벌리고 앉아 핸들을 잡고 패드에 등을 고정해주세요.
  2. 몸통이 흔들리지 않도록 코어(복횡근, 횡경막근, 다열근, 골반저근)에
     힘을 주고 다리를 안쪽으로 모아줍니다.
  3. 위 내용을 15회, 3~5 세트 반복해 줍니다.
